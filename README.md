# Gemini Chatbot

## Project Description

This project is a simple chatbot developed using the Google Gemini API in Python. The chatbot accepts a question from the user and returns an AI-generated response. The project was created and tested in Google Colab and uploaded to GitHub using Git Bash.

## Features

- Simple and beginner-friendly chatbot
- Accepts user input
- Generates responses using the Gemini API
- Developed in Google Colab
- Source code pushed to GitHub using Git Bash

## Technologies Used

- Python
- Google Colab
- Google Gemini API
- Git
- GitHub

## Project Files

- `Gemini_Chatbot.ipynb` – Main chatbot notebook
- `Prompt.txt` – Contains the AI prompts used during development
- `README.md` – Project documentation

## AI Prompts Used

The following prompts were used while developing this project:

1. Create a simple chatbot using the Gemini API key that answers user queries.
2. Create the chatbot in Google Colab using beginner-friendly Python code.
3. Provide the code cell by cell.
4. Explain how to generate and use a Gemini API key.
5. Explain how to deploy the project to GitHub.
6. Explain how to push the project to GitHub using Git Bash instead of uploading files manually.
7. Explain how to securely use the Gemini API key using the Google Colab Secrets environment.
8. Explain how to create `Prompt.txt` and `README.md` files for the project.

## How to Run the Project

1. Open the notebook in Google Colab.
2. Install the required library.
3. Add your Gemini API key (or configure it using Colab Secrets).
4. Run all cells.
5. Enter a question when prompted.
6. The chatbot will generate a response.
